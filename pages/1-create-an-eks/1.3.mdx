## 3. Managed Nodegroup 삭제

이미 구동중이던 Managed Node Group 을 삭제하는 것은 아래의 명령어를 수행하면 됩니다.<br>
<br>

## 노드그룹 삭제
```bash
$ eksctl delete nodegroup --cluster=gitops-study-k8scluster --name=ng-99fc00b4
```
<br>
